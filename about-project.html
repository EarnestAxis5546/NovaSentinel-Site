About NovaSentinel

NovaSentinel is an advanced Anti-DDoS system engineered for robust server protection. Built primarily in C#, it integrates seamlessly with Arch Linux for optimal performance. The architecture focuses on real-time monitoring and mitigation.

Architecture

The system employs a layered approach: ingress traffic filtering, anomaly detection, and response orchestration. C# handles core logic for efficiency in multithreaded environments.

C# Integration

C# is used for the backend services, leveraging .NET for cross-platform compatibility and high-performance networking.

Tech Stack

  • Redis: For caching and real-time data storage
  • Envoy: As a proxy for traffic management
  • eBPF: For kernel-level packet filtering
  • RobustToolbox: Custom toolbox for enhanced security utilities
  • C# / .NET: Core development
  • Arch Linux: Host environment
